# Handwritten Character Recognition

*A machine-learning model that classifies handwritten characters from images.*

## Problem

Digitizing handwriting requires accurately recognizing individual characters from noisy, highly variable image input — a classic computer-vision classification task.

## Approach

Preprocessed and normalized the character images, trained a classification model, and evaluated accuracy on a held-out test set.

## Tech Stack

`Python` · `NumPy` · `pandas` · `Jupyter`

## How to Run

Open `Handwritten_Character_Recognition.ipynb` in Jupyter and run all cells.
